# api-rest-express-typescript-starter

Starter pack for api rest using express and typescript

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install the starter pack.

```bash
npm install
```

## Usage Dev

```
npm run dev
```
## Usage Prod
```
npm run prod
```
